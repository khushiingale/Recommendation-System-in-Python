# 🎯 Recommendation System using Python

## 🚀 Overview
This project builds a recommendation system that suggests items (movies/products) to users based on their preferences and past interactions. It helps users discover relevant content using machine learning techniques.

---

## 🎯 Features
- 📊 Data preprocessing and cleaning  
- 📈 Exploratory Data Analysis (EDA)  
- 🎬 Movie recommendation system  
- 🤖 Content-based and collaborative filtering  
- 🔥 Similarity calculation using cosine similarity  
- 📉 User-item interaction matrix  

---

## 🛠️ Tech Stack
- Python 🐍  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib  
- Seaborn

## ⚙️ How It Works
1. Load dataset (userId, movieId, rating, title)  
2. Create user-item matrix  
3. Apply similarity techniques (cosine similarity)  
4. Use Nearest Neighbors algorithm  
5. Recommend similar items based on user input  

---

## 🤖 Types of Recommendation Systems
- **Content-Based Filtering** → Recommends similar items based on features  
- **Collaborative Filtering** → Recommends based on similar users  

These techniques help predict user preferences effectively.
